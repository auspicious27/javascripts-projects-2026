What We Will Build

Goal: Ek complete responsive and interactive website banana.

Is project mein hum SkillBridge Academy naam ki ek clean educational website banayenge. Website mein navbar, hero section, course cards, feature list, FAQ accordion, contact form, mobile menu, and dark mode hoga. Matlab learner HTML se structure banayega, CSS se design karega, aur JavaScript se page ko interactive banayega.

Story type se samjho: Ek institute ko landing page chahiye jahan students courses dekh sakein, demo class ke liye form fill kar sakein, FAQ open-close kar sakein, mobile par menu open kar sakein, aur dark mode use kar sakein. Yeh real-world frontend website ka beginner version hai.

Website PartTechnologyLearning
Navbar and sectionsHTMLSemantic structure, links, ids, classes.
Cards and responsive layoutCSSFlexbox, Grid, spacing, colors, media queries.
Menu, dark mode, FAQ, form validationJavaScriptDOM selection, events, conditions, functions.

Setup and Run Guide

Goal: Project ko VS Code mein correctly run karna.

  1. VS Code open karo.
  2. Ek folder banao: skillbridge-website.
  3. Folder ke andar 3 files banao: index.html, styles.css, script.js.
  4. HTML code ko index.html mein paste karo.
  5. CSS code ko styles.css mein paste karo.
  6. JavaScript code ko script.js mein paste karo.
  7. Right click on index.html and choose Open with Live Server.
Expected Output: Browser mein professional landing page open hoga. Navbar, hero, course cards, FAQ, contact form and footer dikhega. Mobile size par menu button kaam karega, dark mode button theme change karega, FAQ click par answer open hoga, and form submit par validation message show hoga.

Part 1 - HTML Structure

Goal: Page ka skeleton banana.

HTML website ka structure hota hai. Jaise building mein rooms, doors, windows ka structure hota hai, waise website mein header, nav, main, section, form, footer ka structure hota hai. CSS and JavaScript baad mein isi structure par kaam karte hain.

HTML Code Explanation - Line by Line

Line / CodeDetailed Simple Hinglish Explanation
<!DOCTYPE html>Browser ko batata hai ki yeh HTML5 document hai. Isko file ki first line par likhna best practice hai.
<html lang="en">Complete HTML page ka root element. lang="en" page language English set karta hai.
<head>Browser settings, title, meta tags, and CSS link yahan rakhe jaate hain. Yeh content direct page par visible nahi hota.
<meta charset="UTF-8">Text encoding set karta hai taaki characters properly show hon.
<meta name="viewport"...>Mobile responsive design ke liye important. Page ko device width ke according scale karta hai.
<title>SkillBridge Academy</title>Browser tab mein website ka title show karta hai.
<link rel="stylesheet" href="styles.css">HTML file ko CSS file se connect karta hai. Agar yeh wrong hoga toh design apply nahi hoga.
<body>Visible website content body ke andar hota hai.
<header class="site-header">Top header area. Isme navbar rakha gaya hai.
<nav class="navbar">Navigation section. Logo, links, menu button, theme button yahin aate hain.
<a href="#" class="logo">Website logo link. class="logo" se CSS isko style karega.
<button id="menuButton">Mobile menu open/close button. JavaScript is id se button select karega.
<div class="nav-links" id="navLinks">Navigation links ka wrapper. CSS layout dega, JS mobile par open class toggle karega.
<button id="themeButton">Dark mode toggle button. Click par JavaScript body mein dark class add/remove karega.
<main>Page ka main content wrapper. Header/footer ke beech ka important content yahan hota hai.
<section class="hero">Website ka first main section. User ko page ka primary message yahin dikhta hai.
<p class="eyebrow">Small uppercase label. Project/topic ko visually introduce karta hai.
<h1>Page ka main heading. SEO and readability ke liye important.
<a href="#contact" class="primary-button">Call-to-action link. Click par contact section par jump karega.
<article class="course-card">Independent course card. Har card ek separate content block hai.
<ul class="feature-list">Features ko bullet list mein show karta hai.
<button class="faq-question">FAQ question clickable button. JavaScript click par answer open/close karega.
<p class="faq-answer">FAQ ka hidden answer. JS open class add karega toh answer visible hoga.
<form id="contactForm">Contact form. JavaScript submit event listen karega.
<label for="fullName">Input ka label. for value input id se match honi chahiye.
<input id="fullName" type="text">Name input field. JS yahan se user ka name read karega.
<select id="course">Dropdown field. User course select karega.
<p id="formMessage">Validation message area. JS success/error text yahan show karega.
<script src="script.js"></script>External JavaScript file connect karta hai. Body ke end mein rakha gaya hai taaki HTML pehle load ho jaaye.

Goal: Plain HTML ko professional website design dena.

CSS website ka look and feel control karta hai. Colors, spacing, fonts, cards, navbar, responsive layout, hover states, and dark mode design CSS se banta hai. HTML content deta hai, CSS us content ko attractive and readable banata hai.

JavaScript user actions handle karta hai. Is project mein JavaScript menu open/close karega, dark mode toggle karega, FAQ answers show/hide karega, and contact form validate karega. Simple words mein: HTML page banata hai, CSS page sajata hai, JavaScript page ko react karna sikhata hai.
