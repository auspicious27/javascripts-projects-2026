Module 4

JavaScript Fundamentals - Project Based Notes

Simple Hinglish mein step-by-step project: HTML structure, CSS design, JavaScript logic, DOM updates, events, and form validation.

Project Goal - Hum Kya Build Karenge?

Is module mein hum ek Student Skill Dashboard build karenge. Yeh ek simple but complete interactive web application hai. User form mein student name, email, skill, and practice level enter karega. JavaScript form ko validate karega, invalid input par error show karega, valid input par student ko dashboard list mein add karega, total students count update karega, and reset button se list clear karega.

Is project ko build karte time learners ko JavaScript ke saare important fundamentals practically samajh aayenge. Syntax, variables, data types, conditions, loops, functions, scope, arrays, objects, DOM manipulation, events, and client-side logic ek hi project ke flow mein cover honge.

Class mein aise explain karo: HTML page ka skeleton hai, CSS uska design hai, aur JavaScript us page ka brain hai. Jab user form submit karta hai, JavaScript decide karti hai ki data sahi hai ya nahi, phir page ko dynamically update karti hai.

Step 4 - JavaScript Code

JavaScript ka kaam project ko interactive banana hai. Is file mein hum HTML elements select karenge, array maintain karenge, validation functions banayenge, form submit event handle karenge, and dashboard dynamically update karenge.

const studentForm = document.getElementById("studentForm");
const studentName = document.getElementById("studentName");
const studentEmail = document.getElementById("studentEmail");
const studentSkill = document.getElementById("studentSkill");
const studentLevel = document.getElementById("studentLevel");
const message = document.getElementById("message");
const stats = document.getElementById("stats");
const studentList = document.getElementById("studentList");
const resetBtn = document.getElementById("resetBtn");

let students = [];

function showMessage(text, type) {
  message.textContent = text;
  message.className = "message " + type;
}

function getLevelStatus(level) {
  if (level >= 8) {
    return "Advanced";
  }

  if (level >= 5) {
    return "Intermediate";
  }

  return "Beginner";
}

function renderStudents() {
  studentList.innerHTML = "";
  stats.textContent = "Total Students: " + students.length;

  if (students.length === 0) {
    studentList.innerHTML = '<p class="empty">No students added yet.</p>';
    return;
  }

  for (let student of students) {
    const card = document.createElement("article");
    card.className = "student-card";

    card.innerHTML = `
      <h3>${student.name}</h3>
      <p>Email: ${student.email}</p>
      <p>Skill: ${student.skill}</p>
      <span class="level-badge">${student.status} - Level ${student.level}</span>
    `;

    studentList.appendChild(card);
  }
}

studentForm.addEventListener("submit", function (event) {
  event.preventDefault();

  const nameValue = studentName.value.trim();
  const emailValue = studentEmail.value.trim();
  const skillValue = studentSkill.value;
  const levelValue = Number(studentLevel.value);

  if (nameValue === "") {
    showMessage("Please enter student name.", "error");
    return;
  }

  if (!emailValue.includes("@")) {
    showMessage("Please enter a valid email address.", "error");
    return;
  }

  if (skillValue === "") {
    showMessage("Please select a skill.", "error");
    return;
  }

  if (levelValue < 1 || levelValue > 10) {
    showMessage("Practice level must be between 1 and 10.", "error");
    return;
  }

  const student = {
    name: nameValue,
    email: emailValue,
    skill: skillValue,
    level: levelValue,
    status: getLevelStatus(levelValue)
  };

  students.push(student);
  renderStudents();
  showMessage("Student added successfully.", "success");
  studentForm.reset();
});

resetBtn.addEventListener("click", function () {
  students = [];
  renderStudents();
  showMessage("Student list reset successfully.", "success");
});

renderStudents();

Step 5 - Kaise Run Karna Hai?

  1. VS Code mein student-skill-dashboard folder open karo.
  2. index.html, styles.css, and script.js files create karo.
  3. Upar diya gaya code respective files mein paste karo.
  4. index.html par right click karo and browser mein open karo. Live Server extension hai toh Open with Live Server use karo.
  5. Blank form submit karo: red error message aayega.
  6. Wrong email enter karo: email validation error aayega.
  7. Valid data fill karo: student dashboard mein card add hoga and total count update hoga.
  8. Reset List click karo: array empty hogi and dashboard reset ho jayega.

Expected Output

Browser mein ek clean dashboard dikhega. Top par project title hoga, left/first panel mein registration form hoga, and next panel mein live dashboard hoga. Valid student add karne par card create hoga jisme name, email, skill, and level status show hoga.
